Asnn2 Selenium Tests

Running the Tests

Currently the tests require you to login in as Instructor before running the test, but that will change eventually.

  1. Install Selenium in Firefox from here http://selenium-ide.openqa.org/download.jsp Currently we are using Version 1.0 Beta 2
  2. Login to Sakai as an Instructor and navigate to the Assignments 2 tool on a site in the portal. Open the source of that page.
  3. cd asnn2/selenium-test
  4. Open asnn2_selenium_setup.py. Copy the /portal/* portion of the Assignments2 page into the new_asnn2_page variable so it looks like the current_asnn2_page variable. Search the page source for 'iframe' and copy the name/id of the iframe Main* id into new_asnn2_mainframe so it looks like current_asnn2_mainframe.
  5. Run './asnn2_selenium_setup.py' at the command line while inside the selenium-tests directory. You should see it say that it fixed all the .html files. If it's not executable for some reason, use 'python ./asnn2_selenium_setup.py'
  6. In firefox, still logged in as Instructor, open Selenium IDE from 'Tools -> Selenium IDE'
  7. Choose 'File -> Open Test Suite...' and open 'suite_assignments2.html'
  8. Click the 'Play Entire Test Suite' button. It's right next to the Fast-Slow Slider on the toolbar.
  9. Hopefully it works. You'll see the Runs and Failures as they occur.
